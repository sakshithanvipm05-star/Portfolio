Asura Assist

An embedded AI chatbot that replaces slow, fragmented documentation search with instant, structured answers — reducing product friction at the point of need.

The Problem

Users struggle to find information quickly because knowledge is scattered across tools. Traditional search slows down workflows — people don't want documentation, they want instant answers. Context beats content.

The Solution

  • Embedded AI assistant — ask instead of search
  • Slide-in UI — non-intrusive, always available
  • Bullet-formatted responses — instantly scannable
  • Perceived speed via loader UX
  • End-to-end MVP: Chat UI + FastAPI + OpenAI

What I Built

  • Chat UI — HTML / CSS / JS
  • FastAPI backend with OpenAI integration
  • Error handling & response formatting
  • New chat reset & session management
  • Responsive, clean conversational layout

Key Learnings

  • UX directly impacts perceived AI intelligence
  • Prompting alone is not enough for accuracy
  • Knowledge structure is critical for grounding
  • RAG is the next essential step for production

User Flow — How It Works

STEP 01
User enters query in chat interface
STEP 02
Frontend sends request to FastAPI backend
STEP 03
Backend processes & calls OpenAI API
STEP 04
AI generates structured response
STEP 05
Formatted answer displayed to user
Current Limitations
  • No RAG / knowledge base yet
  • No persistent chat history
  • No context awareness across turns
  • Basic prompt engineering
  • No user authentication
Future Roadmap
  • Implement RAG for grounded responses
  • Add chat memory & context awareness
  • Improve UX — suggestions, streaming
  • Deploy live version with analytics
  • Multi-platform & JIRA integration
